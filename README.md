# Healthcare Analytics Dashboard

## Project Overview
This project analyzes diabetic patient hospital readmissions using SQL, Excel, and Power BI.

## Tools Used
- SQL
- Excel
- Power BI
- DAX

## Key Insights
- Older patients had highest readmission rate
- More inpatient visits increased readmission risk
- Patients not using insulin showed higher readmissions

## Dashboard Features
- KPI cards
- Interactive slicers
- Readmission analysis
- Gender distribution
- Medication usage insights
